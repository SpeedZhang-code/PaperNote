## 1. 用高斯分布建模 GPS 誤差(量測機率 / Emission Probability)

**直覺**

GPS 量測到的位置 $z_t$ 跟車輛實際所在的道路位置 $x_{t,i}$ 之間,一定有誤差。這個誤差是「儀器雜訊」造成的——沒有系統性偏移(不會永遠往東飄或往西飄),而是隨機的、對稱的。

**為什麼選高斯分布(常態分布)?**

物理上,GPS 誤差來自很多獨立小雜訊源疊加(大氣延遲、多路徑反射、衛星幾何誤差…),根據**中央極限定理**,大量獨立小誤差疊加的結果會趨近常態分布。因此「零均值高斯」是一個自然、且被過去研究驗證過的假設。

**公式**

$$p(z_t|r_i) = \frac{1}{\sqrt{2\pi}\,\sigma_z} \, e^{-0.5\left(\frac{\|z_t-x_{t,i}\|}{\sigma_z}\right)^2}$$

白話翻譯:量測點 $z_t$ 跟候選路段 $r_i$ 上最近點 $x_{t,i}$ 的距離越大,這個候選路段是正確答案的機率就越低,且是以高斯曲線的速度下降。

$\sigma_z$ 控制曲線寬度:
- $\sigma_z$ 越大 → 越不信任 GPS 訊號(容許更大誤差)
- $\sigma_z$ 越小 → 要求匹配點必須非常靠近 GPS 點

**關鍵差異(機率模型 vs. 規則式方法)**

傳統方法可能寫死「200 公尺內都算候選」,但 HMM 是用連續的機率分布去加權每個候選,越近權重越高,搭配路網結構後才能做出最優決策。

## 2. 用指數分布刻畫路徑合理性(轉移機率 / Transition Probability)

**直覺**

要判斷「車輛從路段 A 移動到路段 B」是否合理,作者比較兩個距離:

- $\|z_t - z_{t+1}\|_{\text{great circle}}$:兩個 GPS 量測點的**直線距離**
- $\|x_{t,i} - x_{t+1,j}\|_{\text{route}}$:兩個候選路網點之間**實際開車的路徑距離**

若匹配正確,這兩個距離應該差不多(車子開的路程跟 GPS 移動的直線距離不會差太多)。若匹配錯誤,路網距離往往遠大於直線距離(錯誤匹配會導致路徑繞遠路,甚至逆向)。

**為什麼選指數分布,而不是高斯?**

作者用真實資料(ground truth)畫出這個差值的直方圖(論文 Figure 5),發現它長得像指數分布——這是**經驗觀察**,不是先驗假設。

**公式**

$$p(d_t) = \frac{1}{\beta} \, e^{-d_t/\beta}$$

指數分布的關鍵特性:在 0 附近機率最高,然後單調遞減。適合描述「大多數正確匹配的差值都很小,但也允許少數合理的繞路」這種**右偏(right-skewed)**資料型態。高斯分布是對稱的,無法很好描述「不可能是負值、且大多集中在 0 附近」的距離差異資料。

$\beta$ 的意義:
- $\beta$ 越大 → 容忍度越高(允許路徑差異較大也給合理機率)
- $\beta$ 越小 → 要求路徑距離必須很貼近直線距離

## 3. MAD 穩健估計(參數估計 / Robust Parameter Estimation)

**問題**

需要從資料中估計 $\sigma_z$(GPS 雜訊標準差)。最直覺的做法是算標準差,但標準差有個致命缺點:**對離群值(outlier)極度敏感**。GPS 資料常有少數誤差超大的離群點(訊號反射、城市峽谷效應等),這些離群點會讓標準差被嚴重放大,估計出來的 $\sigma_z$ 會失真。

**MAD(Median Absolute Deviation,中位數絕對偏差)怎麼解決這個問題?**

$$\sigma_z = 1.4826 \times \text{median}_t\left(\|z_t - x_{t,i^*}\|\right)$$

步驟拆解:

1. 算出每個 GPS 點跟其正確匹配路段的距離(誤差的估計值)
2. 取這些誤差值的**中位數**,而非平均數——中位數對離群值不敏感(就算有一兩個超大誤差值,只要不超過一半的資料,中位數幾乎不受影響)
3. 乘上校正係數 **1.4826**——目的是讓「資料真的服從常態分布」的前提下,MAD 估計出來的值與真實標準差一致(中位數天生會比標準差小,需要此係數校正回同一尺度)

**核心概念延伸**

這是經典的**穩健統計(robust statistics)**概念:平均數/標準差 vs. 中位數/MAD,後者更能抵抗離群值干擾。此邏輯不僅用於 GPS 誤差,任何有雜訊、有離群值的真實世界資料(如異常監控演算法)都適用同樣原則。

## 三者的邏輯關係總結

| 步驟 | 回答的問題 | 對應理論 |
|---|---|---|
| 1. 量測層面 | 這個 GPS 點,離候選路段多遠算合理? | 高斯分布(雜訊模型) |
| 2. 路徑層面 | 候選路段之間怎麼移動才合理? | 指數分布(路徑合理性模型,由真實資料驗證) |
| 3. 參數估計層面 | 如何從有離群值的真實資料,估計出可靠的 $\sigma_z$? | MAD 穩健估計 |

三者合起來,是這篇論文數據工程的核心精神:**先假設合理的雜訊模型,再用穩健統計方法從真實資料估計參數,最後用機率框架(HMM + Viterbi)整合成最優決策。**
